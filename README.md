Avant de lancer le projet, assurez-vous d’avoir :

XAMPP installé

PHP 8.x

MySQL activé

Redis installé et lancé

Extension PHP Redis activée

Pour lancer le projet:

1/Démarrer Apache et MySQL dans XAMPP

2/Lancer Redis

3/Placer le projet dans : C:\xampp\htdocs\EtuServices

4/Accéder via navigateur : http://localhost/EtuServices
